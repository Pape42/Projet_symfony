Projet Symfony : Gestion de Bibliothèque

Ce projet est une application web pour gérer une bibliothèque.
Il permet de gérer les auteurs et les livres avec un système CRUD (Créer, Lire, Modifier, Supprimer).

Le projet est organisé en plusieurs parties :

Entity : contient les entités Auteur et Livre qui représentent les données de la base.

Controller : contient les contrôleurs qui gèrent les pages et les actions de l’utilisateur.

Form : contient les formulaires Symfony pour créer ou modifier des auteurs et des livres.

Chaque livre est associé à un auteur, ce qui permet de bien gérer les relations entre les données.
L’interface est simple et réalisée avec Symfony et Twig.
